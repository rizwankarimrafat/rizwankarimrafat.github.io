Machine Learning for Robotic Arm and Gripper Control Mar 2024 – Oct 2024

This project investigates advanced machine learning techniques for robotic grasping in unstructured environments, focusing on the Learning to Grasp (L2G) model for 6-DOF grasp prediction using 3D point cloud data. Simulation tests demonstrated a 93.6% grasp success rate and superior efficiency compared to baseline models, showcasing the model's robustness. Real-world deployment on the Doosan A0509 robotic arm revealed challenges such as connectivity issues and hardware limitations, highlighting the complexities of transitioning from simulation to practical applications. Future work aims to enhance perception capabilities and test the model on diverse robotic platforms for broader applicability. Read more

Predictive Modeling and Reinforcement Learning with MiniPong Sept 2024 – Oct 2024

This project focused on applying advanced machine learning and reinforcement learning techniques to the MiniPong gaming simulation. Convolutional Neural Networks (CNNs) were designed for predictive modeling, achieving 89.35% accuracy in multi-output coordinate prediction, while a convolutional autoencoder effectively reconstructed images with minimal loss, showcasing efficient dimensionality reduction. Reinforcement learning agents, including tabular Q-learning and Deep Q-Networks (DQNs), were implemented to optimize decision-making tasks, with the Q-learning agent excelling in Level 1 and the DQN achieving moderate success in Level 3. The project highlights the strengths and challenges of these approaches, offering insights into model performance and opportunities for optimization.

Github: github.com/rizwankarimrafat/Predictive-Modeling-and-Reinforcement-Learning-with-MiniPong

Social Media Network Analysis Using R for #WSUCOMP7025 Hashtag Apr 2024 – May 2024

This project involved conducting a detailed social media network analysis using R on posts tagged with #WSUCOMP7025. Comparative Analysis of C-VAE and C-GAN for Japanese Character Generation May 2024 – Jun 2024

This project explores and compares the performance of Conditional Variational Autoencoders (C-VAE) and Conditional Generative Adversarial Networks (C-GAN) in generating Japanese characters from the Kuzushiji-49 dataset. The models were evaluated on their ability to produce images conditioned on character class and style. The analysis involved both qualitative (visual) and quantitative assessments, including metrics such as Mean Squared Error (MSE) and Structural Similarity Index (SSI). The project provides insights into the strengths and limitations of each model in capturing and replicating fine details in character generation tasks.

Github: github.com/rizwankarimrafat/Comparative-Analysis-of-C-VAE-and-C-GAN-for-Japanese-Character-Generation
